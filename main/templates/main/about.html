О нас

Это сайт для помощи в финансовом учёте, который поможет вам визуализировать ваши доходы и расходы, поставить цель для накопления.

Также на сайте вы сможете получить совет от нейросети на основе ваших трат, доходов и целей.

Технологии, использованные в проекте:

  • Redis: кэширование элементов страницы для сокращения запросов к базе данных и к API курсов валют.
  • Celery, RabbitMQ: асинхронная отправка писем на почту для восстановления пароля.
  • DRF: API для получения информации о пользователях сайта из внешних приложений.
  • PostgreSQL: база данных.
  • Чужой API: получение курсов валют.
  • Django-allauth и social-django: аутентификация пользователей через сайт или соцсети.
  • Docker: контейнеризация.
